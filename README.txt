
Steps to access Menstruation station

git clone https://github.com/hrl2116/Menstruation-Station.git

download node js from https://nodejs.org/en/download/

npm start

To properly view the screens as an app view, please right click in chrome, select “inspect” and then use the device toggle tool bar (cmd+shift+m) to change to a phone view (iPhone 12 Pro)

This app uses the Mapbox API using import mapboxgl
